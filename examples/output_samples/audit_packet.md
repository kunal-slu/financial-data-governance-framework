# Sample Audit Packet

This sample packet shows how multiple generated artifacts can be assembled for review.

Included artifacts:

- validation summary: `validation_summary.json`
- lineage bundle: `lineage_bundle.json`
- compliance summary: `compliance_summary.md`
- model monitoring summary: `model_monitoring_summary.json`

Reviewer intent:

- identify critical rule failures quickly
- confirm lineage completeness
- inspect model-monitoring status
- preserve a compact review trail for the workflow

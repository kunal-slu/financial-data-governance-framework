# v0.1.4 Release Notes

## Summary

This release reflects continued refinement of FDGF as a public reference implementation for governance-as-code patterns in regulated financial data workflows.

## Highlights

- contract-driven validation for regulated financial datasets
- lineage bundle generation and review-oriented artifacts
- illustrative model-monitoring checks
- reusable templates, control sets, and profiles
- improved documentation and artifact consistency

## Notes

This release is intended for inspection, local evaluation, adaptation, and extension. FDGF remains a public reference implementation and not a production platform or regulator-endorsed system.

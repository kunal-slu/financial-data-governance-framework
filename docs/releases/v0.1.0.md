# v0.1.0 Release Notes

## Highlights

- initial public release of FDGF as a governance-as-code reference implementation
- contract-driven validation for regulated financial datasets
- lineage tracking with machine-readable event and bundle outputs
- model-monitoring utilities for drift and fairness checks
- reporting summaries for reviewer-friendly artifact generation
- sample contracts, policies, schemas, and output artifacts

## Included Technical Areas

- financial data validation
- lineage capture
- audit-oriented artifact generation
- model-governance monitoring
- reusable templates and control mappings

## Notes

This release is intended for inspection, local evaluation, and extension. It is a reference framework rather than a completed production platform.
